About the System

Barangay Reserve is a mobile-based system that allows residents to book barangay facilities, manage reservations, and apply authentication-based discounts through a secure platform.

This project was developed as an academic capstone project for facility management in Philippine barangays.

Purpose and Data Collection

We collect user information to:

  • Authenticate residents and verify barangay residency
  • Manage facility bookings for community events and services
  • Process verification requests for official barangay transactions
  • Send notifications about booking confirmations and community updates
  • Maintain communication between residents and barangay officials

🏢 Facility Reservation Scheduling

Book barangay facilities for events, meetings, and community activities with real-time availability checking.

👤 Resident Authentication

Secure online verification process for barangay residents with document upload and status tracking.

🎫 Discount Eligibility Verification

Apply authentication-based discounts for eligible residents through verified user profiles.
